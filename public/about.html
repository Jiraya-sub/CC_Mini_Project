About This Project

This is a beginner-friendly DevOps demonstration project. It showcases how a simple web application can be built, containerized, and deployed using modern software engineering practices.

The project implements a Stocks Dashboard using Node.js and Express, which is then packaged into a Docker container to ensure consistent behavior across different environments.

Technologies Used:

Node.js Express.js Docker GitHub Actions Glassmorphism UI
